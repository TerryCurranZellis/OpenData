
# OpenData Framework

## Overview

The OpenData Framework is an enterprise-grade Java 17 framework for downloading, validating, transforming and importing Open Data into relational databases.

The framework has been designed to be:

- Modular
- Extensible
- Plugin-based
- Database independent
- Fully documented
- Production ready

---

## Key Features

- Java 17
- Maven build
- SQL Server support
- Plugin architecture
- ETL pipeline
- Validation framework
- Dataset metadata
- Scheduling support
- Comprehensive logging
- Documentation-first development
