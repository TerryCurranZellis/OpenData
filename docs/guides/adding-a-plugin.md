# Adding a Plugin

**Document ID:** GUIDE-PLUGIN-001  
**Version:** 1.0  
**Status:** Baseline  
**Baseline date:** 23 July 2026  
**Minimum Java version:** 17

---


Choose a stable lowercase id; create implementation and properties; add id to
index; define endpoint/format/strategy; add typed properties and credential
references; implement only source rules; add registry/configuration tests and
reference documentation. Do not modify `Main` to add a plugin.
