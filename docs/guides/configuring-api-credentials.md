# Configuring API Credentials

**Document ID:** GUIDE-CREDENTIAL-001  
**Version:** 1.0  
**Status:** Baseline  
**Baseline date:** 23 July 2026  
**Minimum Java version:** 17

---


Register the key; store it in a protected provider; commit only a credential
reference; configure endpoint placement; require HTTPS; dry-run with redacted
logging; verify the secret is absent from Git and generated docs.
