# HTML Link Discovery Reference

**Document ID:** REF-HTML-001  
**Version:** 1.0  
**Status:** Baseline  
**Baseline date:** 23 July 2026  
**Minimum Java version:** 17

---


Discovery selects candidate elements with CSS, matches href and optional text,
resolves relative URLs and chooses first/last match. No match is a clear download
failure. Browser automation is outside Phase 1.
