# ADR-0025: Move plugin definitions to database JSON later

**Status:** Shelved  
**Date:** 23 July 2026

## Context

Normalised tables and JSON could centralise administration but add complexity.

## Decision

Keep the design documented; do not implement until properties-based definitions stabilise.

## Consequences

The record model remains storage-neutral and Phase 1 stays simple.
