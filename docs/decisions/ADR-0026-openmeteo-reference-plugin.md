# ADR-0026: Use OpenMeteo as API reference plugin

**Status:** Accepted  
**Date:** 23 July 2026

## Context

A parameterised API example is needed alongside Ofgem.

## Decision

Use the historical archive endpoint with configurable coordinates, dates, timezone and variables.

## Consequences

Exercises query construction and JSON/optional CSV parsing without an API key.
