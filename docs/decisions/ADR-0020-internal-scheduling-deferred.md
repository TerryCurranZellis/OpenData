# ADR-0020: Defer internal scheduling

**Status:** Deferred  
**Date:** 23 July 2026

## Context

Scheduling adds concurrency and operations before manual execution is stable.

## Decision

Use external scheduling for Phase 1.

## Consequences

Core remains focused; internal scheduling is postponed.
