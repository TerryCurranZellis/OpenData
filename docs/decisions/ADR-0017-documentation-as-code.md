# ADR-0017: Maintain documentation as code

**Status:** Accepted  
**Date:** 23 July 2026

## Context

Design evolves with source and must be reviewable.

## Decision

Use Markdown, PlantUML, ADRs and automated builds in the repository.

## Consequences

Docs can be published automatically; stale placeholders are quality defects.
