# ADR-0006: Document every public package

**Status:** Accepted  
**Date:** 23 July 2026

## Context

Package ownership is central to the modular monolith.

## Decision

Maintain package-info.java plus broader Markdown architecture.

## Consequences

Package changes require documentation review.
