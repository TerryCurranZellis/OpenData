# ADR-0014: Separate framework metadata and business tables

**Status:** Proposed  
**Date:** 23 July 2026

## Context

Run metadata is common while dataset schemas differ.

## Decision

Use common execution/artefact tables and plugin-owned business tables.

## Consequences

Operational reporting is consistent; detailed schema awaits implementation.
